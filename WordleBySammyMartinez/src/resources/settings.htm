Command Line Parameters

This code can be ran with the command line parameters: "absurdle" and "random"

How To Run A Java Program With Command Line Parameters

  1. Compile the Program

    Example compilation: javac ProgramName.java

  2. Run the Program and Pass in the Command Line Parameters

    Example of Running with Parameters:

       java ProgramName arg1 arg2 argX....argX

absurdle As Command Line Parameter

Choosing "absurdle" as the command line parameter will launch the game as Absurdle. In this variation of Wordle, there is no hidden word. Instead, each guess will narrow down the list of words into smaller and smaller pools, with each pool based on a guess being chosen by the largest size. This is done to keep the game going as long as possible.

random As Command Line Parameter

Choosing "random" as the command line parameter will randomly launch the game in either Wordle or Absurdle. Both games have a 50% chance of being chosen.
